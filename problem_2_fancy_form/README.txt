Fancy Form: Currency Swap

What this is
- A polished currency swap UI.
- Live pricing from https://interview.switcheo.com/prices.json
- Token icons from https://github.com/Switcheo/token-icons
- Client side validation, mock balances, loading simulation, and success state.

How to run
1. Install Node.js 18+.
2. In this folder:
   - npm install
   - npm run dev
3. Open the URL shown in your terminal.

Build
- npm run build
- npm run preview

Notes
- Tokens with no price are automatically hidden.
- Balances are mocked per token on first load and stored in memory.
- This project uses Vite + React.
